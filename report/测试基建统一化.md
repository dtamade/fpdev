## 遇到的问题与解决
- Release 模式若未设置 SearchPaths，会出现找不到单元错误
  - 解决：在 Release BuildMode 补齐 IncludeFiles/OtherUnitFiles/UnitOutputDirectory
- 历史聚合/遗留工程（tests/ 根部 .lpi/.lpr）未纳入模块化结构
  - 解决：先保证模块化子目录全绿，遗留项后续迁移或单独适配
- 旧 API 测试（test_libgit2_complete）与现封装不一致
  - 解决：最小改动适配当前封装（增加 uses libgit2，用路径与引用判断替代 IsBare/IsEmpty/IsBranch）
