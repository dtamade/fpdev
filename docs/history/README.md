# Historical Documentation

本目录收纳已退出 `docs/` 根目录的历史快照、设计草案与阶段性总结，目的是让公开文档入口保持聚焦，同时保留可追溯的项目演进记录。

## 使用边界

- 当前公开状态、路线图与用户入口：优先查看 `README.md`、`README.en.md`、`docs/ROADMAP.md`
- 当前发布验收：优先查看 `docs/MVP_ACCEPTANCE_CRITERIA.md`、`docs/MVP_ACCEPTANCE_CRITERIA.en.md`
- 历史周报与阶段归档：查看 `docs/archive/`
- 活跃实现计划与 owner checkpoint：查看 `docs/plans/`

## 收录类型

- 历史路线图与早期规划：`DEVELOPMENT_ROADMAP.md`、`DEVELOPMENT_ROADMAP.en.md`、`TODO-FPC-v1.md`
- 历史设计草案：`PACKAGE_CREATION_DESIGN.md`、`PACKAGE_CREATION_DESIGN.en.md`、`PACKAGE_DEPENDENCY_SPEC.md`、`PACKAGE_DEPENDENCY_SPEC.en.md`
- 历史迁移/阶段总结：`PHASE2-MIGRATION-GUIDE.md`、`PHASE2-MIGRATION-GUIDE.en.md`、`PHASE5-SUMMARY.md`、`PHASE6-SUMMARY.md`
- 历史审计/复盘：`ARCHITECTURE_REVIEW.md`、`DEPRECATED_CODE_AUDIT.md`、`B165-i18n-retrospective.md`、`B166-deprecated-cleanup.md`、`B167-cross-split-analysis.md`、`B171-large-files-report.md`、`B172-week11-retrospective.md`
